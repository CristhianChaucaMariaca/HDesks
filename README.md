## BOTONES

### Boton Primario
Para usar los **Botones primarios** se debe incluir clases en las etiquetas `.btn-primary` y así podras usar los estilos.

#### Ejemplo de uso:
```html
<a href="" class="btn-primary">Boton</a>
```
