HDesks

Este proyecto son solo estilos para ser asignados estilo Bootstrap al blog HDesks.com, para mejorar la experiencia de usuario en la navegación. Ademas facilita el manejo de los estilos y asignación de clases al momento de implementar nuevas secciones o estilos.

Las técnologias usadas para este proyecto son SASS para las centralizar las hojas de estilo en un solo archivo y realizar la implementación y modificación de una forma mas organizada y eficiente.
